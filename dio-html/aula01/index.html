Módulo I - Entendendo a comunicação Client x Server

História da Web

Voltar para o início

A World Wide Web (WWW), mais conhecida como Web, é um sistema de hipertexto que permite a usuários acessarem e compartilharem informações através da internet. A Web foi criada em 1989 por Tim Berners-Lee, um físico inglês que trabalhava no CERN, um laboratório de física de partículas na Suíça.

Berners-Lee desenvolveu a Web como uma maneira de compartilhar informações entre os pesquisadores do CERN. Ele criou o HTML (HyperText Markup Language), uma linguagem de marcação que permite a criação de páginas da Web que podem conter links para outras páginas. Ele também criou o HTTP (HyperText Transfer Protocol), um protocolo de comunicação que permite a transferência de páginas da Web entre computadores.

A World Wide Web foi lançada ao público em 1991 e rapidamente se tornou popular. Em 1994, o número de sites na Web já ultrapassava 10.000. Hoje, a Web é uma parte essencial da vida cotidiana de bilhões de pessoas em todo o mundo.

"A World Wide Web não é apenas uma tecnologia, é uma maneira de viver." - Tim Berners-Lee

O que são Clients

Voltar para o início

Um cliente é um programa de computador que se comunica com um servidor para solicitar um serviço. Por exemplo, quando você acessa um site na internet, seu navegador é o cliente que se comunica com o servidor do site para solicitar a página.

Os clientes podem ser instalados em computadores, smartphones, tablets e outros dispositivos. Eles podem ser usados para acessar uma variedade de serviços, como e-mail, redes sociais, bancos, compras online e muito mais.

O que são Servers

Voltar para o início

Um servidor é um computador que fornece um serviço para outros computadores. Por exemplo, um servidor web pode fornecer páginas da Web para clientes que acessam um site.

Os servidores podem ser instalados em computadores, servidores dedicados ou na nuvem. Eles podem ser usados para fornecer uma variedade de serviços, como e-mail, redes sociais, bancos, compras online e muito mais.

Linguagens de Programação

Voltar para o início

Uma linguagem de programação é uma linguagem usada para escrever programas de computador. Existem muitas linguagens de programação diferentes, cada uma com suas próprias vantagens e desvantagens.

Algumas das linguagens de programação mais populares usadas para desenvolver sites incluem JavaScript, Java, PHP, Python e Ruby.

Módulo II - Introdução ao HTML na Prática

Fundamentos e ferramentas utilizadas

Voltar para o início

O HTML (HyperText Markup Language) é uma linguagem de marcação usada para criar páginas da Web. O HTML é usado para definir a estrutura e o conteúdo de uma página da Web.

O que é o inspetor de Elementos

Voltar para o início

O inspetor de Elementos é uma ferramenta que permite inspecionar o código HTML de uma página da Web. Ele pode ser usado para ver como os elementos da página são estilizados e para fazer alterações no código HTML.

Ele está disponível na maioria dos navegadores da Web e para acessá-lo, você pode pressionar F12 no teclado ou clicar com o botão direito do mouse em um elemento da página e selecionar "Inspecionar".

Qual a estrutura básica do HTML

Voltar para o início

A estrutura básica do HTML é a seguinte:

<!DOCTYPE html>
<html>
<head>
      <title>Título da página</title>
</head>
<body>
      Conteúdo da página
</body>
</html>

  • O elemento <!DOCTYPE html> declara que o documento é um documento HTML.
  • O elemento <html> é o elemento raiz do documento.
  • O elemento <head> contém informações sobre a página, como o título e os metadados.
  • O elemento <title> define o título da página.
  • O elemento <body> contém o conteúdo da página.

Sobre Tags e seus atributos

Voltar para o início

As tags são usadas para marcar o conteúdo de uma página da Web. Cada tag tem um nome e pode ter atributos.

Os atributos são usados para fornecer informações adicionais sobre a tag. Por exemplo, o atributo "href" da tag <a> é usado para especificar o URL do link ou navegação na mesma página com o scroll automático para aquele item ou seção.

Textos, Listas e Links

Voltar para o início

O HTML fornece várias tags para formatar texto, criar listas e criar links.

  • Textos: As tags <h1>, <h2>, <h3>, <h4>, <h5> e <h6> são usadas para criar cabeçalhos. As tags <p> são usadas para criar parágrafos e <br> para quebras de linha.
